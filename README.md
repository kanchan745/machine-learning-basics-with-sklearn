Machine Learning Basics with Scikit-Learn

This repository is created to understand and practice the fundamentals of Machine Learning using Python and scikit-learn.

Topics Covered

 1. Classification Algorithms
- Logistic Regression (Binary Classification)
- K-Nearest Neighbors (KNN)
- Decision Tree Classifier
- Real-world examples like pass/fail prediction and fruit classification

 2. Model Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
- Regression Metrics: MAE, MSE, RMSE

 Tools & Libraries Used
- Python
- scikit-learn
- NumPy

Purpose
This project is for learning and practicing machine learning concepts in a simple and easy-to-understand way.  
Suitable for beginners and students starting with ML.
